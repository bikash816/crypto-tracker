# crypto-tracker
A CLI-based crypto price tracker built with Python. Fetches live prices, market cap, and 24h changes for BTC, ETH, USDT, BNB, XRP, SOL ,USDC,TRX, STRTH, AMD DOGE using the CoinLore public API. built as my first real python API project.

have live price alert option that will notify when the coin reached the target and get notify in the terminal as well as at telegram.
